The 'emnist' folder contains the binary version of the EMNIST-letters dataset, downloaded
from here: https://www.nist.gov/itl/products-and-services/emnist-dataset


The 'cmnist' folder contains the binary (only black and white) version of the CMNIST dataset.
The .csv files define the training, testing and validation splits and the different organizations
of the dataset:
- Russian
- Cyrillic Letters Unbalanced
- Cyrillic Letters Balanced
- Cyrillic Lowercase and Uppercase Unbalanced
- Cyrillic Merged Unbalanced
'cmnist' folder also contains the original paper which describes the dataset.
You can also see the paper here: https://aclanthology.org/2022.lrec-1.510.pdf
You can download the dataset from: https://github.com/bolattleubayev/cmnist
